# Job Split API Documentation
## 1 List JobSplits
- **Endpoint**: `GET /jobs/{job_id}/job_splits`
- **Summary**: 获取指定任务的子任务列表 (Get a list of subtasks for a specified task)
- **Description**: 获取指定任务的子任务列表 (Get a list of subtasks for a specified task)
### Request Parameters
- **Path**:
  - `job_id` (integer, required, minimum: 1): 任务ID (Task ID)
